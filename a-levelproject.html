A-Level Computer Science Project

A project made for my final year A-Level Computer Science course. The project is a virtual reality stealth based game in which you have to avoid cameras, lasers and basic enemy AI. The main challenge to overcome when making the game was learning how to create a game specifically based for VR. Movement works in a completely different way to that of a typical first person game, interacting with the environment and objects works in a completely different way. So it created an interesting learning opportunity to work out how developing games worked in VR.
